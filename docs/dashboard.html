What is this?

A knowledge graph of San Francisco emergency services, built to demonstrate when graph databases add value over flat tabular approaches.

Integrates SFFD fire/EMS and SFPD police dispatches through a shared ontology — two independent datasets unified by semantic alignment.


Data sources

  • SF Fire Dept Calls for Service — 3.4M records (2003–2026)
  • SFPD Dispatched Calls — 7.6M records (2014–2026)
  • Graph filtered to 2020+ → 10.5M nodes in FalkorDB
